基础：

布莱丝的物理攻击倍率与工作倍率均为0.84。
布莱丝的光束/导弹/超级导弹不受上述倍率影响。
点击屏幕左下角的按钮可以查看技能和魔法。

---------------------------------------------------------------
技能/魔法获取指南：

宽光束：
使用能换取金子的物品与猪王交易（例如，交给猪王一块大肉）获得。

波动光束：
击败梦魇疯猪或者拾荒疯猪获得。

导弹：
地上世界会随机刷新大约10个带有导弹柜的彩蛋。直接拾取导弹柜并使用即可解锁。
离出生点最近的导弹柜相关彩蛋会刷新在混合区地形上。
解锁导弹后，你就可以自己在制作栏随时制作导弹了。
解锁导弹后，再次使用导弹柜可以提升导弹上限。
此外，如果你运气很差，直到探远古时也没能解锁导弹，也可以在远古伪科学站处解锁。

超级导弹：
打败远古档案馆深处的圣地守卫，或在打败蝙蝠洞地形深处的恶液巨物戈尔戈罗斯获得。
其他性质与导弹类似。

速度推进器：
打败远古守护者（犀牛）获得。

闪光转移：
在地底月光蘑菇林的一个鸟人雕像处获得。

其他的技能/魔法尚未制作获取途径。
---------------------------------------------------------------
新增的boss：

圣地守卫：
固定刷新在远古档案馆的远古小合奏机位置。
普通攻击力低（约为20伤害/拳），
但是在非防御形态下，会不间断地使用超高伤害的猛扑攻击。
圣地守卫说出“摧毁 摧毁 摧毁”同时眼睛发光就是准备猛扑的前兆。
圣地守卫在承受一定伤害后进入防御形态，此时它每被次攻击都会立刻进行反击，
圣地守卫在持续防御形态一小段时间后会进行一轮原地敲地板攻击并退出防御形态。
防御形态下的圣地守卫并没有增加防御力。
每当圣地守卫退出防御形态，让它再次进入防御形态所需的伤害都会提高。

恶液巨物戈尔戈罗斯：
刷新在地底蝙蝠洞地形的一个区域。
只会通过跳跃与喷射恶液团进行攻击，
喷射的恶液团落地后会形成小片减速区域。
伤害很高，且具有位面伤害。自身具有位面实体抵抗（会减免物理伤害）且免疫位面伤害。
但是十分笨重，喷射恶液团的精度也很差。建议使用远程攻击。

---------------------------------------------------------------
新增的天气：

落星雨：
每天傍晚和晚上，天上会随机掉下坠落之星。
4个坠落之星（烤熟的坠落之星也行）入锅可以制作魔力水晶，可用于增加饥饿上限。

雷暴：
夏天会随机发生。表现为频繁的雷击与骤雨。
